# Personal Fitness Tracker Dashboard

# Project Overview

The Personal Fitness Tracker Dashboard is a Python-based project that helps users record, manage, and analyze their daily fitness activities. It allows users to track workouts, calculate fitness metrics, and visualize performance trends over time.
* Log daily fitness activities (Running, Cycling, Yoga, etc.)
* Calculate total calories burned
* Compute average workout duration
* Filter activities by type
* Generate detailed statistical reports
* Visualize data using:

  * Bar Charts
  * Line Graphs
  * Pie Charts
  * Heatmaps

* **Python**
* **Pandas** → Data handling and analysis
* **NumPy** → Numerical computations
* **Matplotlib** → Data visualization
* **Seaborn** → Advanced visualization

## Dataset Information

The dataset contains the following columns:

* **Date** → Date of activity
* **Activity Type** → Type of workout
* **Duration (minutes)** → Time spent
* **Calories Burned** → Energy burned

## Project Objective

The goal of this project is to build a simple and effective system for tracking fitness activities and analyzing health data using Python and data visualization tools.
